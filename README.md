# TitanicSurvivalModel_Analysis
This project builds and compares 3 machine learning models to predict 
passenger survival on the Titanic using the Kaggle Titanic dataset.

Source: Kaggle Titanic Competition

##Steps I Covered
1. Data Loading & Exploration
2. Data Preprocessing & Cleaning
3. Feature Engineering
4. Model Training (3 Models)
5. Model Evaluation & Comparison
6. Best Model Analysis & Conclusion

 ##Models Used
LogisticRegression,RandomForest,KNN.
 Best Model
Random Forest & Logistic Regression tied with 79.33% accuracy.
Random Forest selected as final best model.

 ##Tools Used
- Python, Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
- Google Colab
